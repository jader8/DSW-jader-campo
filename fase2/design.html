fundamentos css

segun(santos D 2022) CSS son las siglas en inglés para «hojas de estilo en cascada» (Cascading Style Sheets). Básicamente, es un lenguaje que maneja el diseño y presentación de las páginas web, es decir, cómo lucen cuando un usuario las visita. Funciona junto con el lenguaje HTML que se encarga del contenido básico de las páginas.

frameworks css

segun m,l.(2021) Los frameworks CSS son herramientas que contienen colecciones de propiedades de estilo, generalmente diseñadas en archivos conocidos como hojas de estilo. Los frameworks son el ingrediente principal del desarrollo web robusto, llevando fácilmente a una interfaz de usuario atractiva y funcional.
